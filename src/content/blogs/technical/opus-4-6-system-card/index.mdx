## 概述

Claude Opus 4.6 是 Anthropic 于 2026年3月6日发布的旗舰级大语言模型，首个支持 1M token 上下文窗口的 Opus 级模型。本文深度解读其官方 System Card 的安全评估与技术细节。

## 核心能力

### 1. Adaptive Thinking

动态推理深度选择，根据查询复杂度调整思考过程。

### 2. Context Compaction

长上下文压缩功能，优化大文档处理效率。

### 3. Effort 控制

4 级控制选项：low / medium / high / max
