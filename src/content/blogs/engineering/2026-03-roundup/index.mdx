## 概述

2026年3月是 Claude 模型的重大发布月份，Anthropic 推出了 Opus 4.6 和 Sonnet 4.6，均支持 1M token 上下文窗口。同时 DeepMind 更新了 Gemini 3.1 系列。

## Anthropic 重要发布

### Claude Opus 4.6

- **来源**: [Anthropic News](https://www.anthropic.com/news/claude-opus-4-6)
- **日期**: 2026-03-06
- **核心内容**:
  - 首个 1M token 上下文窗口的 Opus 级模型
  - Terminal-Bench 2.0 最高分
  - Humanity's Last Exam 领先
  - GDPval-AA 比 GPT-5.2 高 144 Elo 分
  - BrowseComp 最佳表现
  - SWE-bench Verified 81.42%（优化后）
  - 支持 128k 输出 token
  - Context Compaction 功能
  - Adaptive Thinking
  - 4 级 Effort 控制（low/medium/high/max）
  - API 价格：$5/$25 per million tokens

### Claude Sonnet 4.6

- **来源**: [Anthropic News](https://www.anthropic.com/news/claude-sonnet-4-6)
- **日期**: 2026-03-06
- **核心内容**:
  - Pro/Free 用户默认模型
  - 1M token 上下文窗口
  - OSWorld 计算机使用基准显著改进
  - 支持 Adaptive Thinking 和 Context Compaction
  - $3/$15 定价
  - 70% 用户偏好于 Sonnet 4.5
  - 59% 用户偏好于 Opus 4.5（因较少过度工程和"懒惰"）
  - Vending-Bench Arena 展现长期规划能力

**合作伙伴验证**:
- Notion（复杂请求执行）
- Replit（agentic workflows）
- Sourcegraph（并行子代理）
- Windsurf（调试和代码库理解）
- Box（金融推理 QA 提升 10%）
- Lovable（设计系统）
- Figma（Figma Make）
