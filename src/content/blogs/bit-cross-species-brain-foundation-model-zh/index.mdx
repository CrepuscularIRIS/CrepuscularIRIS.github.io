语音 BCI 多年来遵循一条固定的流水线：记录神经活动、解码音素、将音素送入语言模型、输出文本。每个阶段独立优化。语言模型补偿音素解码器的错误，但无法将梯度信息回传以改善解码器。这种级联架构能工作——BrainGate 团队和其他研究组已经构建了令人印象深刻的实时系统——但它对性能设置了硬性上限，因为每个阶段的误差是累积而非抵消的。

BIT（BraIn-to-Text），在 ICLR 2026 上由 Zhang 等人发表，打破了这条流水线。论文引入了两个关键创新：一个通过自监督学习在跨物种和跨任务数据上预训练的神经编码器，以及一个将该编码器直接连接到音频 LLM 进行文本生成的端到端架构。结果是相比之前最佳端到端方法，词错误率（WER）降低了 58%，从 24.69% 降至 10.22%。

## 级联架构的天花板

要理解端到端为什么重要，需要看清标准流水线的实际工作方式。

典型的语音 BCI 从植入在腹侧前运动区和中央前回的 Utah 微电极阵列进行记录。记录捕获两种信号类型：阈值化脉冲计数（单单元活动）和脉冲带功率（多单元群体活动）。这些信号以 20ms 窗口分箱，产生一个随时间变化的神经特征矩阵。

级联解码器通过三个独立阶段处理这个矩阵：

```
神经活动 → [音素解码器] → 音素序列
                              ↓
               [n-gram 语言模型] → 候选句子
                              ↓
                  [重打分/束搜索] → 最终文本
```

每个箭头都是一道硬边界。音素解码器训练目标是最小化音素分类错误。语言模型在文本语料上训练。两者无法影响对方的训练过程。如果音素解码器经常混淆 /b/ 和 /p/，语言模型学会绕过这种混淆——但音素解码器从未收到"区分这两个音素对句子级准确率更重要"的信号。

最好的级联系统在尝试性语音上达到约 6-9% 的 WER，这很了不起。但端到端训练可以做得更好，因为它让编码器学习针对最终目标（正确文本）优化的表征，而非中间代理任务（正确音素）。

## 跨物种预训练

BIT 的第一个贡献是一个预训练的神经编码器。这遵循深度学习中现已标准化的范式：在大规模多样化数据集上用自监督学习预训练，然后在目标任务上微调。

关键在于：预训练数据来自人类和猴子。

编码器是一个 Transformer，以时间 patch 为单位处理神经记录。每个 patch 覆盖多个 20ms 时间窗，捕获神经放电模式的时间动态。预训练目标是 masked patch modeling——随机遮挡时间 patch，训练编码器从周围上下文重建它们。

预训练数据集：367 小时的 Utah 阵列记录，跨越：

- **人类语音数据** — 瘫痪参与者产生句子的尝试性语音记录
- **人类运动数据** — 手臂和手部运动记录
- **猴子运动数据** — 恒河猴的抓取和伸够任务记录

两个设计决策值得关注。

**跨物种迁移确实有效。** 猴子运动皮层数据改善了人类语音解码。这并非显而易见——猴子皮层有不同的层状组织、不同的电极放置位置，任务也完全不同（手臂运动 vs 语音产生）。但预训练目标迫使编码器学习神经群体动力学的通用原则：神经元群体如何随时间协调活动、运动规划和执行期间神经轨迹如何展开。这些动力学跨物种保守，因为运动皮层共享进化结构。

消融实验清晰地证明了这一点：

```
编码器配置                    | WER (级联) | 改进
BIT-TFS (从零训练)            | 基线       | —
BIT-Human (人类数据预训练)     | 更好       | 跨任务有帮助
BIT-All (人类+猴子数据预训练)   | 最好       | 跨物种有帮助
```

**主题特异性适配器处理变异性。** 每个记录主体获得一个小型线性投影层，将其电极配置映射到共享编码器空间。这处理了电极放置、阻抗和信号质量在不同主体和植入会话之间差异巨大的事实。共享编码器学习通用神经动力学；主体特异层处理测量噪声。

## 端到端架构

第二个贡献将预训练的神经编码器连接到音频 LLM 进行直接句子生成。

作者使用的类比是 LLaVA：LLaVA 通过连接图像编码器给语言模型一双"眼睛"，BIT 通过连接神经编码器给语言模型一颗"大脑"。

具体模型：Aero1-Audio 1.5B，Qwen2.5-1.5B 的音频扩展版本。架构：

```
神经活动 → [预训练 BIT 编码器] → 神经嵌入
                                     ↓
                         [对比学习对齐]
                                     ↓
              音频 LLM (Aero1-Audio 1.5B) → 文本输出
```

对比对齐步骤至关重要。神经嵌入和音频嵌入处于不同的表征空间。对齐训练使用对比学习将神经嵌入映射到对应语音的音频嵌入附近，创建一个共享空间，让 LLM 可以像处理退化音频一样处理神经输入。

端到端系统使用组合损失训练：对比损失用于对齐，加语言建模损失用于文本生成。梯度从文本输出一路回传，通过对齐层进入神经编码器。这意味着编码器学习产生对最终任务（生成正确句子）有用的表征，而非中间代理。

## 结果

Brain-to-Text 基准测试上的性能：

| 系统 | 架构 | WER |
|------|-----|-----|
| 之前 SOTA (级联) | 音素解码器 + 5-gram LM | ~6.5% |
| 之前 SOTA (端到端) | 直接 seq2seq | 24.69% |
| BIT (级联) | 预训练编码器 + n-gram LM | 新 SOTA |
| BIT (端到端) | 预训练编码器 + Aero1-Audio 1.5B | 10.22% |

10.22% 的端到端 WER 仍高于最好的级联系统（~6.5%），但差距已经大幅缩小。端到端系统在扩展时具有结构性优势：更多数据和更大的 LLM 应该持续改善性能，而级联系统因为每个阶段独立优化会遇到递减收益。

## 内部语音迁移

最具前瞻性的结果：BIT 的预训练编码器对齐了尝试性语音嵌入与想象性语音嵌入。

尝试性语音是瘫痪参与者试图说话时产生的——运动皮层像产生语音一样放电，但信号没有到达肌肉。想象性语音是参与者在不试图产生语音的情况下想象说话。其神经特征更弱且更不明显。

BIT 表明，在尝试性语音数据上预训练然后在少量想象性语音数据上微调的编码器，可以解码想象性语音。跨任务迁移有效是因为预训练编码器已经理解了语音运动规划的时间动态——想象性语音激活相同的网络，只是振幅降低。

如果这一发现可推广，它开辟了一条通向无需用户尝试发声即可解码内在思维的 BCI 路径。那是一类不同的设备——不再是语音假体，而更接近思维接口。

## 架构启示

BIT 验证了几个超越 BCI 领域的模式：

**基础模型预训练对神经数据有效。** 自监督范式（遮挡与重建）从文本和图像迁移到了神经记录，尽管信号的噪声和变异性大了几个数量级。关键要素是拥有足够多样的数据来学习通用动力学，而非主体特异的噪声模式。

**跨域数据改善目标域性能。** 猴子运动数据帮助人类语音解码，是 ImageNet 预训练帮助医学影像的神经科学等价物。当底层动力学共享结构时，跨域预训练提供有用的归纳偏置。

**数据和计算充足时端到端训练优于级联流水线。** 这与端到端语音识别（用 Whisper 风格模型替换 HMM-GMM-LM 级联）应用于神经信号的经验一致。

**小型 LLM 足够。** BIT 使用 1.5B 参数的 LLM，而非 70B 前沿模型。对于当前 BCI 数据集中有限的词汇和句子结构，一个带有良好神经输入的小模型比一个带有差神经输入的大模型更有效——这对边缘设备部署具有实际意义。

## 下一步

当前的瓶颈是数据。367 小时的皮层内记录按神经科学标准已经很多——它代表了多个研究站点数年的患者会话。但按深度学习标准来看，这微不足道。将 BIT 扩展到数千小时的神经数据、纳入更多物种、更多皮层区域和更多任务类型，应该能显著改善预训练编码器。

更长远的问题是这种架构能否扩展到非侵入性记录模态。Utah 阵列提供高质量信号是因为它们位于皮层内部。EEG 位于颅骨外部，捕获的信号是模糊的、衰减的、混合的。基础模型方法——在海量多样化数据上预训练，在目标任务上微调——有可能通过纯粹的数据规模克服 EEG 的信号质量限制。但目前还没有人展示过这一点。

BIT 论文表明，改变了 NLP 和计算机视觉的范式——基础模型、自监督预训练、端到端训练、规模化——同样适用于神经接口。语音 BCI 正在遵循与语音识别相同的轨迹：从级联专家系统到端到端学习系统，基础模型提供关键的预训练步骤。
