几十年来，神经科学对视觉意象有一个工作假设：当你想象一个物体时——一张脸、一个地标、一个场景——大脑会招募感知时激活的同一批神经群体，但振幅降低，空间精度下降。这个"再激活假说"得到了 fMRI 研究的支持，这些研究显示意象期间的视觉皮层 BOLD 信号与感知期间的相似。但 fMRI 测量的是数百万神经元和数秒时间平均的血氧水平。它无法告诉你单个神经元在意象期间是否与感知期间做同样的事情，也无法判断群体层面的相似性是否掩盖了根本不同的单细胞计算。

2026年4月9日发表于《Science》的一项研究，由加州理工学院的 Cerf、Kreiman、Koch 及其同事完成，在单个神经元水平上解决了这个问题。他们利用植入癫痫患者颞叶内侧（MTL）的微电极阵列，在受试者同时观看和想象特定视觉类别时记录单个神经元活动。核心发现：在感知期间使用分布式轴向编码——而非稀疏的"祖母细胞"响应——来编码类别身份的同一神经元，在纯粹的心理意象中被重新激活，神经状态空间中的几何组织相同。

这不是对我们已知信息的确认。这是一个改变神经解码系统可靶向目标的机制性结果。

## 分布式轴向编码

关键的概念贡献是描述了单个 MTL 神经元如何编码视觉类别。传统模型——珍妮弗·安妮斯顿一个神经元、埃菲尔铁塔一个神经元，每个神经元对"自己的"刺激最大程度放电——是祖母细胞假说。它一直是一个简化，但 MTL 中的单神经元记录曾提供了一些最强有力的证据：单个神经元对特定面孔、地点和物体具有选择性反应，跨越视角、大小和呈现格式的剧烈变化。

加州理工学院团队表明，类别编码更好地描述为**分布式轴向编码**。每个神经元不是一个类别的检测器。相反，每个神经元占据一条连续响应轴上的一个位置，该轴跨越多个类别，而群体的联合活动定义了一种几何结构——神经状态空间中的一系列轴——其中每个类别对应一个不同的方向。

```python
import numpy as np
from dataclasses import dataclass


@dataclass(frozen=True)
class NeuralAxisCode:
    """MTL神经元中类别编码的几何表示。

    每个类别是 N 维空间中的一个方向（轴），
    该空间由 N 个记录的神经元定义。
    神经元对类别的贡献是其在该类别轴上的载荷。
    """
    n_neurons: int
    n_categories: int
    axes: np.ndarray       # (n_categories, n_neurons): 类别轴
    responses: np.ndarray  # (n_trials, n_neurons): 记录的活动

    def decode_category(self, trial_activity: np.ndarray) -> int:
        """从单次试验的神经活动中解码类别。

        将活动投影到每个类别轴上。
        投影最大的轴即为解码的类别。
        """
        projections = self.axes @ trial_activity
        return int(np.argmax(projections))

    def reconstruction_similarity(
        self, perceived: np.ndarray, imagined: np.ndarray,
    ) -> float:
        """测量感知状态和意象状态之间的几何相似性。

        如果意象复用相同的轴向编码，
        同一类别的感知向量和意象向量之间的夹角应该很小（高余弦相似性）。
        """
        norm_p = np.linalg.norm(perceived)
        norm_i = np.linalg.norm(imagined)
        if norm_p == 0 or norm_i == 0:
            return 0.0
        return float(np.dot(perceived, imagined) / (norm_p * norm_i))
```

祖母细胞编码与分布式轴向编码之间的区别对解码有影响，因为它们具有不同的信息论特性。祖母细胞编码用 N 个神经元编码 N 个类别，每个神经元携带单个刺激的类别特异性信息。分布式轴向编码用 K 个神经元编码 N 个类别（可能 K < N），每个神经元携带多个类别的部分信息。轴向编码更高效（每个类别需要的神经元更少）、更鲁棒（失去一个神经元略微降低所有类别而非完全摧毁一个类别）、更容易解码（线性分类器有效，因为类别边界是神经空间中的超平面）。

## 意象复用了感知几何

核心实验中，受试者在交替的区块中执行两个任务。在感知任务中，他们观看来自多个视觉类别（面孔、地标、物体、动物）的图像，同时记录单神经元活动。在意象任务中，他们收到类别名称的提示，并被要求生动地想象该类别中的一个特定范例——闭着眼睛，没有视觉输入。

发现：感知期间观察到的轴向编码几何在意象期间得以保留。具体来说：

**相同的轴。** 神经状态空间中的类别方向在感知和意象之间对齐。如果神经元 A 在感知期间对"面孔"轴有正向贡献，那么在意象期间它也对"面孔"轴有正向贡献。同一类别的感知轴和意象轴之间的余弦相似度显著高于随机打乱对照组的值，而跨类别比较接近零。

**振幅降低。** 意象期间群体反应的幅度约为感知反应的 60-70%。几何结构相同；信号更弱。这与 fMRI 再激活文献一致，但现在在单神经元水平上得到确认。

**辨别力保持。** 尽管振幅降低，类别边界仍保持线性可分。在感知数据上训练的分类器可以解码意象试验，反之亦然。跨条件解码准确率低于同条件内准确率，但明显高于随机水平。

```python
from typing import NamedTuple


class DecodingResult(NamedTuple):
    within_perception: float
    within_imagery: float
    cross_perception_to_imagery: float
    cross_imagery_to_perception: float
    chance_level: float


def illustrative_results() -> DecodingResult:
    """研究解码准确率的大致模式。

    同条件内解码准确率最高。
    跨条件解码（在感知上训练，在意象上测试）
    较低但远高于随机水平。
    不对称性（感知→意象 vs 意象→感知）
    反映了意象期间的振幅降低。
    """
    return DecodingResult(
        within_perception=0.89,
        within_imagery=0.76,
        cross_perception_to_imagery=0.71,
        cross_imagery_to_perception=0.68,
        chance_level=0.25,  # 4 个类别
    )
```

跨条件迁移是 BCI 最看重的结果。这意味着在校准了感知响应（可以在受控的刺激驱动会话中收集）的解码器将泛化到意象响应。用户不需要在校准期间进行想象。他们需要看图片。意象解码器是免费附赠的。

## 这对神经解码意味着什么

解码想象内容的 BCI 系统面临一个根本性的校准问题。你不能像问"你在看什么？"那样，用同样的实验控制来问"你在想象什么？"意象是自愿的，在生动性上存在差异，且难以验证。每个解码想象的 BCI 系统都必须处理其训练信号——真值标签——存在噪声这一事实。

轴向编码结果提供了一种绕过方法。如果感知表征和意象表征共享相同的几何结构，你可以在感知上校准（真值干净），然后部署在意象上（真值不可用）。意象期间的振幅降低是一个缩放因子，而非结构性变化。归一化了振幅的解码器应该能够迁移。

这在三个层面有实际意义：

**更快的校准。** 当前的运动意象 BCI 需要 30-60 分钟的校准，用户反复想象特定动作。如果运动皮层使用类似的轴向编码结构——犹他阵列记录有证据表明确实如此——校准可以使用被动观看运动视频代替，缩短会话时间，减少用户疲劳。

**更丰富的词汇表。** 意象期间可解码的类别数量受限于神经状态空间中可区分方向的数量。轴向编码框架给出了几何界限：你可以编码与记录群体中线性独立方向数量一样多的类别，减去噪声边界。对于 100 个调谐良好的神经元，这可能是几十个类别——远超过当前 BCI 通常支持的 2-4 个运动意象类别。

**隐蔽交流。** 从 MTL 神经元解码想象视觉类别的 BCI 可以支持闭锁患者的无声交流。患者想象一系列类别-对象，每个映射到一个字母或单词。解码器读取轴向编码激活并输出文本。不需要运动输出。

## MTL 记录约束

注意事项是记录方式。加州理工学院的研究使用颞叶内侧的微电极阵列——为临床癫痫监测放置的植入式深部电极。这是有创神经外科手术。研究中的患者已经因为医疗原因植入了电极；研究记录是顺势而为。

对于消费级 BCI 产品，MTL 记录不可行。问题是轴向编码结构是否存在于非侵入性方法可触及的信号中：

**EEG。** 头皮 EEG 没有单神经元分辨率。但如果轴向编码是群体层面的几何特性，它可能可以从聚合信号中恢复。EEG 源定位方法（波束成形、LCMV）可以估计特定脑区的活动，而 MTL 等深部源最难从头皮测量中恢复。头皮 EEG 中 MTL 活动的信噪比很差，但不为零——多项研究已经从 EEG 中以高于随机的准确率解码了粗略的类别信息（面孔 vs 地点 vs 物体）。轴向编码几何是否在体积传导的空间模糊中存活，是一个开放的经验问题。

**ECoG（皮层电图）。** 放置在皮层表面的硬膜下电极网格比 EEG 有更好的空间分辨率，但不能直接到达 MTL。然而，皮层视觉区域（V1-V4、IT）可能对视觉类别实现类似的轴向编码。颞皮层上的 ECoG 可能通过足够的电极覆盖来恢复意象几何。

**立体脑电图（sEEG）。** 为癫痫监测放置的深部电极——与加州理工学院研究中使用的相同方式——作为诊断工具正变得越来越常见。一些 BCI 研究人员开始开发专门用于解码的慢性可植入 sEEG 阵列，具有更小的电极轮廓和无线数据传输。这是将轴向编码结果转化为临床 BCI 的最直接途径，但手术门槛仍然存在。

| 记录方式 | 单神经元 | MTL  접근 | 侵入性 | BCI 可行性 |
|---|---|---|---|---|
| 微电极阵列 | 是 | 直接 | 高 | 仅研究 |
| 立体脑电图 (sEEG) | ~LFP | 直接 | 中-高 | 临床试验 |
| ECoG | 否 | 间接 | 中 | 临床试验 |
| 高密度 EEG | 否 | 很差 | 无 | 消费级（受限）|
| MEG | 否 | 中等 | 无 | 仅实验室 |

## 基础模型关联

轴向编码结果以特定方式与 BCI 基础模型文献相关联。像 BrainGPT、BIT 和 PanLUNA 这样的模型通过在大规模 EEG/fMRI 数据集上的自监督预训练来学习脑信号表征。然后学习到的表征被微调用于下游任务——情绪识别、睡眠分期、运动意象分类。

如果大脑的类别表征具有轴向编码几何，那么一个从噪声记录中学习恢复这种几何的基础模型，应该比学习平坦特征空间的模型实现更好的迁移学习。轴向方向是表征的自然基——它们是大脑使用的坐标。一个与这种几何对齐的基础模型学习新类别时需要更少的微调数据，因为每个新类别是已学习空间中的一个新方向，而非无结构的全新特征。

这类似于学习解纠缠表征的视觉 transformer 比学习纠缠表征的迁移更好。轴向编码是"解纠缠"对神经表征意味着什么的生物学先验：每个轴是一个独立的类别方向，表征沿着这些轴分解。

对于 BCI 基础模型研究人员，实际含义是一种新的预训练目标：从原始神经信号中学习恢复轴向编码几何。如果模型的习得表征保留了类别轴之间的角度关系，它就捕捉到了可以从感知迁移到意象的结构——使跨条件解码成为可能的结构。

## 值得关注

三个进展将决定轴向编码结果是否从《Science》论文转化为已部署的 BCI 能力：

**慢性 sEEG 试验。** 下一步是从具有慢性植入物的患者记录 MTL 神经元，持续数周或数月，而非癫痫监测期间可用的数天。慢性记录将显示轴向编码是随时间稳定的还是漂移的，以及反馈训练是否能提高意象类别解码准确率。

**非侵入性轴恢复。** 如果高密度 EEG（256+ 通道）配合先进源定位能够从 MTL 活动中恢复即使退化的轴向编码几何，消费者 BCI 路径就会打开。信号质量会更差，但几何结构——如果存在的话——比原始频谱特征提供更强的解码先验。

**基础模型对齐。** 使用轴向编码感知目标训练的 BCI 基础模型是否在意象解码任务上优于使用标准自监督目标训练的模型。一旦轴向编码几何被形式化为训练目标，这是一个直接的实验。

加州理工学院的结果是对该领域二十年来一直在绕圈子的问题的一个清晰答案：大脑在单神经元水平上复用了感知代码来想象吗？确实如此。相同的轴，相同的几何，振幅降低。对于任何构建神经解码器的人来说，含义是直接的：在用户所看之物上校准，在用户所想象之物上部署。几何可以迁移。
